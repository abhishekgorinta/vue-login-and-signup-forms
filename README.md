# Login and Signup form using vue

A small Vue 3 single-page application scaffolded with Vite. It includes basic routing and authentication-related components (Login, Signup, Home) and a shared `Navbar` component.

## ✨ Features
- Simple Vue 3 + Vite setup
- Client-side routing with `vue-router`
- Components: `Home`, `Login`, `Signup`, `Navbar`

## 🛠 Tech stack
- Vue 3
- Vite
- vue-router

## 📁 Project structure
- public/ — static assets
- src/
	- App.vue
	- main.js
	- style.css
	- assets/
	- components/
		- Home.vue
		- Login.vue
		- Signup.vue
		- Navbar.vue
	- routes/
		- index.js
- index.html
- package.json



## 🚀 Install dependencies:
Getting started
Prerequisites: Node.js (v16+) and npm.

```
npm install
```

Start development server:

```
npm run dev
```
